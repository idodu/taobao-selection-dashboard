# 淘宝家清个护 / 日化纸品选品看板

这是一个静态可视化选品系统，用于每天展示家清个护、日化纸品方向的10个候选品，包含商品图片、各平台参考价格、公开销量/评论信号、利润口径和1-10分综合热度评分。

## 运行方式
- 本地预览：在仓库根目录运行 `python -m http.server 8080`，访问 `http://localhost:8080`。
- 云端运营：GitHub Actions 工作流 `.github/workflows/daily-update.yml` 每天 `02:00 UTC` 运行，即北京时间 `10:00`。
- 发布方式：GitHub Pages 使用 Actions 部署，电脑关机也不影响。

## 评分规则
综合热度分为1-10分，生成脚本位于 `scripts/generate_daily_data.py`。

总分 = 跨平台热度35% + 价格竞争力25% + 利润可行性20% + 销量信号15% + 上架可操作性5% - 风险扣分。

- 跨平台热度：淘宝/天猫、京东、抖音、小红书出现榜单、热词、内容种草或行业报告信号越多，得分越高。
- 价格竞争力：建议到手价越接近主流成交低位，且不牺牲毛利，得分越高。
- 利润可行性：按 `建议到手价 * 78%` 倒推成本上限，能保留15%-25%毛利的商品得分更高。
- 销量信号：公开月销、评论量、榜单名次、内容互动等信号越强，得分越高。
- 上架可操作性：经销现货、标准品牌品、组合装/赠品差异化空间越大，得分越高。
- 风险扣分：控价、重货运费、液体破损、资质/功效合规、品控投诉等会扣分。

## 数据维护
- 商品池：`data/product_catalog.json`
- 每日生成结果：`data/recommendations.json`
- 每日归档：`docs/daily/YYYY-MM-DD.md`

如果有真实供货价，可把实际采购、快递和保质期信息补入商品池，评分会更贴近实际利润。
