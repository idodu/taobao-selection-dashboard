# 淘宝家清个护 / 日化纸品每日选品执行模板

## 固定口径
- 店铺阶段：新店从0起步。
- 货源类型：经销现货，品牌正品优先。
- 目标毛利：最低15%-25%。
- 每日输出：10个候选品，其中标注3个最建议当天上架。
- 成本口径：采购含税价 + 打包耗材 + 实际快递/运费险，不得高于表内成本上限。
- 成本上限估算：建议到手价 * 78%，其中预留15%最低毛利、7%平台/优惠/退货损耗。

## 每日筛选流程
1. 热度池：观察淘宝/天猫、京东、抖音、小红书至少2个平台信号。
2. 价格池：对标同品牌同规格或近似规格，到手价控制在主流成交价低位。
3. 货源池：只保留现货可供、凭证完整、保质期安全、履约稳定的SKU。
4. 上架池：每天10个候选品按3个引流款、5个平销款、2个利润款分层。

## 复盘节奏
- 72小时：看曝光、点击、收藏加购，无曝光先改标题和主图。
- 7天：看转化、退款、毛利，低效品淘汰或降为活动品。
- 14天：保留10-20个稳定SKU，形成小货盘。
- 30天：明确主推品、利润品、活动品。
